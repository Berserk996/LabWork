Классы эквивалентности и граничные значения, условие:
1. От 500 грн до 999 грн — 5%
2. От 1000 грн до 1999 грн — 10%
3. От 2000 грн до 2999 грн — 15%
4. Более 3000 грн — 20%

Классы эвквивалентности:
1. 0 до 499 грн - нет скидки
2. 500 до 999грн - 5%
3. 1000грн до 1999грн - 10%
4. 2000грн до 2999грн - 15%
5. 3000грн до и дальше - 20%

Граничные значения:
(допустим "x" число в(не) диапазона) 
1. x > 500 - не допустимые значения для 5% скидки 
2. 500 <= x <= 999 - допустимые значения для 5% скидки
3. 1000 < x недопустимые значения для 5% скидки

4. x < 1000 -  недопустимые значения для 10% скидки
5. 1000 <= x <= 1999 - допустимые значения для 10% скидки
6. 1999 <= x - недопустимые значения для 10% скидки

7. x < 2000 - недопустимые значения для 15% скидки
8. 2000 <= x <= 2999 - допустимые значения для 15% скидки
9. 2999 < x - недопустимые значения для 15% скидки

10. x < 3000 - недопустимые значения для 20% скидки
11. 3000 <= x - допустимые значения для 20% скидки
